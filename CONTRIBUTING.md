# Contributing to ControlBook

¡Gracias por tu interés en contribuir a ControlBook! 🎉

## Cómo Contribuir

### 1. Fork el Proyecto
```bash
git clone https://github.com/TU_USUARIO/controlbook.git
cd controlbook
```

### 2. Crear una Rama Feature
```bash
git checkout -b feature/mi-nueva-caracteristica
```

### 3. Hacer Cambios
- Sigue el código existente
- Mantén consitencia con el estilo

### 4. Commit
```bash
git commit -m "feat: agregar nueva característica"
```

### 5. Push
```bash
git push origin feature/mi-nueva-caracteristica
```

### 6. Pull Request
- Describe claramente los cambios
- Incluye screenshots si aplica

## Tipos de Contribuciones Welcome

- 🐛 Reportar bugs
- ✨ Nuevas características
- 📚 Documentación
- 🎨 Mejoras UI/UX
- ⚡ Performance improvements
- 🧪 Tests

## Directrices

1. **Código**: Sigue las convenciones existentes
2. **Commits**: Usa mensajes descriptivos
3. **Tests**: Asegúrate que todo funciona
4. **Docs**: Actualiza documentación si es necesario

## Código de Conducta

- Sé respetuoso
- Sé inclusivo
- Sé profesional

¡Esperamos tu contribución! 🚀
